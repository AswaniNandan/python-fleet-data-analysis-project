
### python-fleet-data-analysis

## Overview
This project demonstrates end-to-end data analysis using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Workflow
- Loaded raw dataset
- Cleaned data (removed null values, duplicates)
- Performed analysis using groupby
- Created visualizations

## Key Insights
- Some departments have higher equipment count
- Equipment distribution is uneven
